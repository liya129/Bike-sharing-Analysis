# Bike Sales Analysis Dashboard

## Project Summary
This project presents an in-depth analysis of bike sales data using Microsoft Excel.  
The main objective was to study customer purchasing behavior and identify important sales trends through data visualization and dashboard creation.

---

## Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning Techniques
- Interactive Dashboard

---

## Work Performed
- Organized and cleaned raw sales data
- Removed duplicate and missing values
- Created pivot tables for detailed analysis
- Built charts and dashboard for visualization
- Analyzed customer income, age, and commute patterns

---

## Major Findings
- Customers with higher annual income purchased more bikes
- Middle-aged individuals showed higher buying interest
- Shorter commute distance increased bike purchase probability
- Married customers contributed significantly to overall sales
